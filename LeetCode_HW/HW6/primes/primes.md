# Primes
Given an integer n, return the number of prime numbers that are strictly less than n.

**Example 1:** <br>

Input: n = 10 <br>
Output: 4 <br>
Explanation: There are 4 prime numbers less than 10, they are 2, 3, 5, 7. <br>

**Example 2:** <br>

Input: n = 0 <br>
Output: 0 <br>

**Example 3:** <br>

Input: n = 1 <br>
Output: 0 <br>
 

**Constraints:** <br>

0 <= n <= 5 * 106 <br>
